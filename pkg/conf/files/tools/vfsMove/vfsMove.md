Moves or renames a file or directory from the source path to the destination path.

Usage:
- The path parameter must be an absolute path, not a relative path
- The destination parameter must be an absolute path, not a relative path
- This tool will move the file or directory from path to destination
- If the destination is an existing directory, the source will be moved into that directory
- If the destination does not exist, the source will be renamed to the destination path
- Use with caution as this operation will overwrite the destination if it exists and is a file
